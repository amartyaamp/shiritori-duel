# ⚔️ Shiritori Duel: The Snob's Gambit

Welcome to the digital arena of **Shiritori Duel**, a game where your vocabulary determines your survival.

## 📜 The Legend
One fine afternoon feeling sleepy, **Amartya** thought of the super anime *'No Game No Life'* and envisioned a world where disputes were settled via word-chains rather than coffee. This app is the result of that vision.

## 🎮 How to Play
- **The Chain:** Match the last letter of your opponent's word.
- **The Rules:** 3+ letters. No Proper Nouns (Sorry, Steve). No Stop Words.
- **The Score:** You start at 100. Be the first to reach 0 by submitting words quickly. If the timer goes negative, your score goes up!

## 🛠️ Technical Spells
- **Frontend:** Vanilla HTML5, CSS3, and JavaScript.
- **Multiplayer:** P2P connection via [PeerJS](https://peerjs.com/).
- **Dictionary:** Real-time validation via [Free Dictionary API](https://dictionaryapi.dev/).
- **Deployment:** Automatically deployed via Vercel.
